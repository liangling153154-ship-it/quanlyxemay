
# Sổ thuê xe

Ứng dụng quản lý thuê xe máy cho homestay. Theo dõi lượt thuê, công nợ với công ty Minh Quý, kho xe, và xuất báo cáo dạng ảnh.

## Mở app

https://liangling153154-ship-it.github.io/quanlyxemay/

## Tính năng

- Quản lý lượt thuê: trạng thái (cần thanh toán / dự kiến / đã thanh toán / hủy), nửa ngày, ghi chú xe, tên khách
- Ghi nhận chi phí phát sinh (sửa chữa, bảo dưỡng) tự động trừ vào tổng phải trả
- Kho xe rảnh tại homestay
- Thống kê tháng: số lượt thuê, ngày thuê, doanh thu, ứng trước
- Xuất báo cáo dạng ảnh PNG có ảnh xe, gửi Zalo trực tiếp
- Sao lưu / khôi phục dữ liệu qua file JSON

## Cài đặt lên màn hình điện thoại

1. Mở link bên trên trong Chrome / Safari
2. Menu → "Thêm vào màn hình chính"
3. Mở từ icon như app thường

## Dữ liệu

Lưu trên thiết bị (localStorage). Hãy bấm "Tải file sao lưu" trong Cài đặt định kỳ để tránh mất dữ liệu khi xóa cache trình duyệt.
